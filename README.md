## ðŸ†˜ Troubleshooting

### App Won't Start
- Check macOS version (10.15+ required)
- Try restarting your Mac
- Reinstall from fresh download

### Poor Recognition Quality
- Check microphone permissions in System Preferences
- Test microphone in other apps
- Reduce background noise
- Speak more clearly and slowly

### Auto-Paste Not Working
- Grant accessibility permissions in System Preferences > Security & Privacy
- Restart the app after granting permissions
- Check that target application accepts text input

### Can't Sign In
- Check internet connection
- Try signing out and back in
- Clear browser cache if using web sign-in
